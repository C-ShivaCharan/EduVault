# EduVault – Centralized Academic Library

A full-stack academic platform built with Python and Flask that centralizes learning resources for students.

## Features

- 📚 Access to research papers and e-books in one place
- 🧠 Quiz-based self-paced learning modules
- 🔍 Search and filter resources by subject or category
- 📱 Responsive UI for seamless experience across devices

## Tech Stack

| Layer      | Technology                        |
|------------|-----------------------------------|
| Backend    | Python, Flask                     |
| Frontend   | HTML, CSS, JavaScript             |
| Database   | SQL                               |

## Getting Started

### Prerequisites
- Python 3.8+
- pip

### Installation
```bash
git clone https://github.com/C-ShivaCharan/eduvault.git
cd eduvault
pip install -r requirements.txt
python app.py
```
